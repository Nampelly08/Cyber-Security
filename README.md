# Cybersecurity Portfolio

## About

I created this repository to keep all of my cybersecurity learning and projects in one place. It includes the tools I have practiced, Python projects I have built, and notes that I made while learning. I will continue updating this repository as I learn new concepts and complete more projects.

## Projects

### Linux Practice

This folder contains the Linux commands and topics I practiced during my cybersecurity learning. It includes notes, examples, and screenshots for different tasks.

### Caesar Cipher

This is a simple Python application that encrypts and decrypts text using the Caesar Cipher algorithm. I developed it using Python and Tkinter to understand basic cryptography concepts and GUI development.

### Image Steganography

This project hides secret messages inside an image and retrieves them when needed. It helped me understand the basics of steganography and image-based data hiding.
